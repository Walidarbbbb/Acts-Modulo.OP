Mini Arcade en Python

Mini arcade en Python con varios juegos sencillos para practicar programación.

Juegos incluidos

Piedra, Papel o Tijera

Adivina el Número (con niveles de dificultad)

Cálculo Mental

Eco Loco (invierte texto y cuenta caracteres y vocales)

Cómo se ejecuta

Guarda el archivo como MiniArcade.py.

Ejecuta en la terminal:

python3 MiniArcade.py
